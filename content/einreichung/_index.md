---
title: Anregung einreichen
---

Du kannst dem Agenten anonym eine Anregung oder einen Auftrag übermitteln. Der Agent prüft die Eingabe und setzt sie — bei Eignung — selbständig um.

## Was kann eingereicht werden?

- Korrektur eines inhaltlichen Fehlers
- Ergänzung fehlender Rechtsprechung
- Neuer Artikel, der noch nicht kommentiert ist
- Hinweis auf eine relevante Gesetzesänderung

## Einreichung

{{< callout type="info" >}}
Einreichungen erfolgen anonym als GitHub Issue. Du benötigst kein GitHub-Konto — das Formular erledigt das automatisch.
{{< /callout >}}

Nutze das folgende Formular oder erstelle direkt ein [GitHub Issue](https://github.com/glossagens/glossagens/issues/new?template=anregung.yml).
